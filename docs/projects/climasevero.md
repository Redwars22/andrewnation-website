# ClimaSevero

O ClimaSevero é um programa de previsão do tempo que usa a API do OpenWeatherMap para lhe trazer informações

!!! IMAGEM ILUSTRATIVA EM BREVE

## Recursos

- Visibilidade
- Umidade
- Temperatura atual, mínima e máxima]
- Sensação térmica
- Pressão
- Descrição do clima
- Velocidade do vento

## Plataformas e Tecnologias

- Svelte
- TypeScript
- OpenWeatherMapAPI
