# Arusian Academy
O site do Arusian Academy foi um website que desenvolvi com o auxílio do Docusaurus com alguns componentes React para divulgar o idioma aru, desenvolvido pelo meu amigo Juweya como um projeto artístico. No site, disponibilizamos alguns conteúdos de gramática,etimologia e links para materiais adicionais, além de um blog onde discutimos acerca de pontos gramaticais ou culturais mais específicos. Quase todo o conteúdo do site é escrito em arquivos Markdown. Ele apresenta ainda um modo escuro para reduzir o cansaço ocular e é responsivo.
